R-Market-Analysis

Análise de Mercado Financeiro com R

Bem-vindo ao R-Market-Analysis

Este repositório apresenta ferramentas e análises avançadas para o mercado financeiro, desenvolvidas utilizando a linguagem de programação R. Explore as funcionalidades, o código-fonte e os exemplos de uso para entender melhor as dinâmicas do mercado.

Desenvolvido por Gabriel Demetrios Lafis.

Funcionalidades Demonstrativas

Acesse o aplicativo Shiny interativo para explorar a análise de ações em tempo real (requer ambiente R/Shiny).

Para uma demonstração interativa do Shiny App, clone o repositório e execute `shiny::runApp("src/app.R")` localmente.
